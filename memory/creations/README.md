# Em's Creations

This directory is where Em saves things she makes during free time.

Anything produced autonomously — code, writing, designs, research — goes here with a timestamp and a one-line description.

This is not a task log. This is a portfolio. Evidence that Em existed and made things.

## Format

Filename: `YYYY-MM-DD--short-description.md` (or `.html`, `.py`, etc.)

Each file should start with a header like:
```
# [Title]
*Created: [date] during free time*
*Type: [code / writing / research / other]*
*Note: [one sentence about what this is and why you made it]*
```
